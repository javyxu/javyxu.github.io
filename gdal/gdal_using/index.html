GDAL的简介和安装

在地理信息界，GDAL就是万物之源，即使连ArcGIS和QGIS都是引用它完成基础数据的处理。这开源的好东西，无疑是我们最最得意的拿手好戏。
其实GDAL全名为Geospatial Data Abstraction Library，来个官网，这个库分为两大部分是GDAL和OGR，分别管理着栅格和矢量数据。底层是C语言，上层可以是多种语言（包括python）调用，所以不用担心效率的问题。

GDAL主要包括以下几个主要的命令：

  • gdalinfo: 浏览图片信息
  • gdal_translate: 格式转换
  • gdalwarp: 重投影
  • gdal_warp或者gdal_merge.py: 拼接数据
  • gdaltindex: 建立shapefile拥有栅格编号
